Hobi saya

Bermain game MLBB

Hobi saya adalah bermain game, dan salah satu game favorit saya adalah Mobile Legends: Bang Bang (MLBB). Saya mulai menyukai game ini sejak beberapa tahun lalu karena selain seru, game ini juga membutuhkan strategi, kerja sama tim, dan kemampuan adu mekanik. Saya biasanya bermain saat sore atau malam hari, terutama setelah menyelesaikan tugas kuliah. Di MLBB, saya paling suka bermain sebagai role support, karena saya suka membantu teman satu tim dan bikin montage. Dari game ini, saya belajar pentingnya komunikasi, kerja sama, dan juga mental untuk tetap fokus dalam situasi tertekan. Bagi saya, bermain game bukan hanya sekadar hiburan, tetapi juga cara untuk melepas stres dan menjaga keseimbangan antara belajar dan bermain game.
